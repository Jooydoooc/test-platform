# Project Brief

Platform name: Lexora

Lexora is a private English practice and progress platform for my own students.

The platform focuses mainly on General English, not IELTS.

Lexora helps students:
- practise English with teacher-selected materials
- complete textbook-based exercises
- learn vocabulary
- take grammar, reading, listening, and vocabulary tests
- see their results
- review mistakes
- track progress
- compete with classmates
- stay motivated

Lexora helps the teacher:
- manage students
- organize groups
- upload tasks and tests
- add textbook-based practice
- check student results
- track class progress
- motivate students through ranks, badges, and leaderboards

Lexora is not a public website. It is a closed platform only for my students.

The platform should feel:
- professional
- modern
- clean
- motivating
- organized
- student-friendly
- serious but not boring

It must not look like a childish quiz website, a public course website, or a generic AI-generated platform.
